# План редизайна лендинга Lumi

Цель: привести сайт к новому ASO-позиционированию (voice-first + автоматизация; кластеры couples/shared, separate accounts, multi-currency) и поднять конверсию в клик по App Store.

## Принципы

Один месседж на первом экране — голос. Это самый понятный и дифференцирующий крючок: его нет ни у одного из 18 конкурентов из вашего анализа, и он теперь в subtitle стора («Voice Money Manager, Auto Log»). Всё остальное (Back Tap, Apple Pay, прогноз) — поддерживающие механизмы, а не заголовок.

Показывать, а не рассказывать. У вас уже есть готовые анимации в money-tracker (`lumi-intro-animation.html`, `lumi-scan-animation.html`, `lumi-autopilot-animation.html`, `lumi-goals-animation.html`, `lumi-currencies-animation.html` и др.) — они должны работать вместо текста.

Полная согласованность со стором. Один и тот же hook, те же фичи в Free/Premium, тот же тон. Пользователь идёт лендинг → стор; любое расхождение в момент решения снижает конверсию.

## Новая структура (9 секций вместо 13)

1. **Hero — voice-демо.** Новый заголовок уже внедрён: «Coffee, five euros.» / Logged. Добавить анимацию: произносимая фраза → появляющаяся транзакция с категорией. CTA: App Store + QR. Бейджи доверия: No bank login · Data on device · Free.
2. **How it works (3 шага).** Объединить с текущей Apple Pay-секцией: «Скажи / Заплати / Готово» — голос, автоимпорт, Back Tap как три пути одной идеи «ввод исчезает».
3. **Forecast.** ИИ-прогноз — вторая по силе фича: «видишь конец месяца за 7–10 дней». Использовать `lumi-insights-animation.html`.
4. **Для пар (новая секция).** Кластер couples/shared теперь в 20+ локалях ASO, на лендинге его нет вообще. Совместный бюджет, раздельные счета, общие цели. Использовать `lumi-goals-animation.html`.
5. **Privacy + Comparison (объединить).** Таблица сравнения уже сильная — добавить в неё строку shared budget. Privacy-блок как часть таблицы (Bank login: Never).
6. **Отзывы.** Заменить Reddit-цитаты на реальные отзывы из App Store (по локалям). Если отзывов мало — честный заголовок «Что люди говорят о трекерах расходов» + ваш ответ, как сейчас, но с явной маркировкой.
7. **Pricing.** Уже согласован со стором (Apple Pay auto-import перенесён в Premium). Проверить Back Tap: в сторе он в блоке Premium, в коде приложения — нет пейволла. Решить и синхронизировать.
8. **FAQ (сокращённый, 4–5 вопросов).**
9. **Финальный CTA + футер.** Blog preview убрать с главной (оставить в футере) — он размывает финальный оффер.

## Дизайн-направление

- Тёмный hero с крупной типографикой цитаты («Coffee, five euros.») — цитата как визуальный приём сразу объясняет продукт.
- Меньше секций-карточек, больше «одна секция = одна анимация телефона + один тезис».
- Sticky mobile CTA оставить (уже есть).
- Скриншоты/фреймы переиспользовать из `screenshot-generator` в money-tracker для консистентности со стором.

## Копирайт: правила для локалей

- Обращение: ru/uk — всегда «вы», de — всегда «du» (уже консистентно), остальные — как сейчас.
- Не оставлять английские единицы: sec, mo (исправлено во всех локалях).
- Цены: подставлять валюту storefront, если появится динамика; пока единый формат `$4.99/<мес. локализовано>`.
- SEO title/description теперь содержат voice во всех 10 локалях (внедрено).
